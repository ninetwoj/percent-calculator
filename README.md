---

## 📊 퍼센트 계산 기본 공식

- 비율 계산 = (값 ÷ 전체) × 100  
- 증가율 = ((새 값 - 기존 값) ÷ 기존 값) × 100  
- 할인 가격 = 원가 - (원가 × 할인율)
